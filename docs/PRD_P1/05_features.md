# GCP P1 — 기능 명세서

## 1. 인증 / 회원

### 1.1 회원가입

| 항목 | 내용 |
|------|------|
| 경로 | `/register` |
| 사용자 | 비로그인 |
| 설명 | 학생 또는 코치 역할을 선택하여 계정 생성 |

**기능 상세**
- 역할 선택 탭 (학생 / 코치)
- 이메일, 닉네임, 비밀번호, 비밀번호 확인 입력
- 주 게임 + 티어 선택
- 이메일 중복 시 해당 필드 아래에 `이미 회원가입이 되어있는 이메일입니다.` 표시
- 닉네임 중복 시 해당 필드 아래에 `이미 존재하는 아이디입니다.` 표시
- 성공 시 로그인 페이지로 이동

### 1.2 로그인

| 항목 | 내용 |
|------|------|
| 경로 | `/login` |
| 사용자 | 비로그인 |

**기능 상세**
- 이메일 + 비밀번호 입력
- Enter 키로 로그인 가능 (onKeyDown)
- 아이디 저장 체크박스 (localStorage에 이메일 저장/삭제)
- 비밀번호 오류 시 이메일은 유지, 비밀번호만 초기화
- 에러 메시지는 ✕ 버튼으로만 닫힘 (자동 사라짐 없음)
- 성공 시 유저 정보를 Zustand + localStorage에 저장 후 메인으로 이동

**P1 로그인 응답 처리**
```javascript
// P1: res.data = { id, email, nickname, role, ... } (토큰 없음)
setUser(res.data)
```

### 1.3 로그아웃

- Navbar의 로그아웃 버튼 클릭
- localStorage에서 user, token 제거
- Zustand user 상태 null로 초기화

### 1.4 프로필 수정

| 항목 | 내용 |
|------|------|
| 경로 | `/profile` |
| 사용자 | 로그인 (P1: 보호 없음) |

**기능 상세**
- Canva 스타일 인라인 편집 레이아웃
- 항목별 편집 버튼 클릭 시 해당 필드만 펼쳐짐
- 닉네임, 이메일(수정불가), 주게임/티어, 프로필 색상, 비밀번호 변경
- `PUT /api/users/:id` 호출

## 2. 강의

### 2.1 강의 목록

| 항목 | 내용 |
|------|------|
| 경로 | `/lectures` |
| 사용자 | 전체 (공개) |

**기능 상세**
- 게임별 카테고리 탭 필터 (전체/LOL/발로란트/TFT/배틀그라운드/오버워치/스타크래프트)
- 가격 범위 슬라이더 (실시간 동적 표시)
- 티어 드롭다운 필터
- 포지션 드롭다운 필터
- 코치 유형 뱃지 필터 (프로게이머/프로코치/스트리머/할인중)
- 키워드 검색 (강의 제목 + 코치명)
- 정렬: 랭킹순/평점순/가격낮은순/가격높은순/최신순
- 복합 필터 동시 적용
- 강의 카드 클릭 시 상세 페이지 이동
- 최근 본 강의 섹션 (최대 5개, localStorage)

### 2.2 강의 상세

| 항목 | 내용 |
|------|------|
| 경로 | `/lectures/:id` |
| 사용자 | 전체 (공개) |

**기능 상세**
- 강의 정보 (제목, 게임, 티어, 가격, 코치 정보)
- 평균 평점 + 수강생 수 + 리뷰 수
- 비로그인 시 `로그인 후 수강 신청이 가능합니다.` 안내
- 신청 버튼 상태: 수강 신청 → 검토 중 → 수강 중 / 거절됨
- 이미 신청한 강의 재신청 시 토스트 표시
- 승인된 수강생/코치에게만 ▶ 강의 수강하기 버튼 노출
- 리뷰 목록 표시

### 2.3 강의 등록 / 수정

| 항목 | 내용 |
|------|------|
| 경로 | `/coach/lecture/new`, `/coach/lecture/edit/:id` |
| 사용자 | 코치 (P1: 보호 없음) |

**기능 상세**
- 강의 제목, 설명, 가격, 원래 가격(선택), 대상 티어, 포지션, 코치 유형 입력
- `POST /api/lectures` 또는 `PUT /api/lectures/:id`

### 2.4 강의 콘텐츠 (수강생 시청)

| 항목 | 내용 |
|------|------|
| 경로 | `/lectures/:id/contents` |
| 사용자 | 승인된 수강생 / 코치 (P1: 보호 없음) |

**기능 상세**
- 좌측 목록 사이드바 (order_num 순)
- YouTube iFrame 플레이어 (embed URL 자동 변환)
- 콘텐츠별 댓글

### 2.5 강의 자료 관리 (코치)

| 항목 | 내용 |
|------|------|
| 경로 | `/lectures/:id/manage` |
| 사용자 | 코치 (P1: 보호 없음) |

**기능 상세**
- YouTube URL 입력 시 자동 embed 변환
- 콘텐츠 제목, 설명, 순서 설정
- CRUD 관리

## 6. UI/UX 공통 기능

### 6.1 다크 모드
- 우측 상단 토글 버튼으로 라이트/다크 전환
- localStorage 퍼시스트 (새로고침 후에도 유지)
- Tailwind CSS `dark:` 유틸리티 클래스 적용

### 6.2 로딩 / 빈 상태
- `LoadingScreen` 컴포넌트: 데이터 로딩 중 표시
- `EmptyState` 컴포넌트: 데이터 없을 때 안내 + 액션 버튼

### 6.3 상태 관리 (Zustand)
| 스토어 | 관리 상태 |
|--------|----------|
| useAuthStore | user, token (로그인 정보) |
| useLectureStore | 강의 필터, 검색 상태 |
| useThemeStore | 다크/라이트 모드 |
